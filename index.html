3D CSS3 Playground (Early)

This is an experimental CSS3 3D-Playground, should work in newish versions of Chrome and Safari.

Don't Panic!

Stage Text:   Background-Image:   Text-Color:    Show Grid:

Font-Size: 30px

Text Top: 50%

Text Left: 50%

Perspective: 600

Rotate X: 0 deg

CSS


#stage 
{ 
  -webkit-perspective: 600;
  -webkit-perspective-origin-x: 50%;
  -webkit-perspective-origin-y: 50%;
}

#stage p
{
  font-size: 30px; 
  top: 50%;
  left: 50%;
  -webkit-transform: rotateX(0deg) rotateY(-90deg) rotateZ(0deg) skewX(0deg) skewY(0deg);
  
}
